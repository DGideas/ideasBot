# About ideasBot 2.0
ideasBot用于操作MediaWiki。

ideasBot 2.0已经经过修订，对于面向对象的操作进行优化，还增加了对于DGStorage的原生支持。

# ideasBot 2.0 的更改
* 我们不再使用```api.php?format=xml```的方式解析返回的xml数据。转而使用```json```格式。
* 拓展的使用方式发生了更改，取消了```db```,```lab```和```extension```目录。
